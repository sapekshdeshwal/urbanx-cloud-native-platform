# Docker Day 1 - Interview Questions

## Q1. What is Docker?

Docker is an open-source containerization platform used to package applications along with their dependencies into lightweight and portable containers.

---

## Q2. Why Docker?

Docker solves the "Works on My Machine" problem by providing a consistent runtime environment.

---

## Q3. What is Docker Hub?

Docker Hub is a public registry used to store and share Docker Images.

---

## Q4. Difference between Docker Image and Container?

Image is a blueprint.

Container is the running instance of an Image.

---

## Q5. What is Nginx?

Nginx is a high-performance web server that serves web pages and can also act as a reverse proxy and load balancer.

---

## Q6. What is docker exec?

It executes commands inside a running container.

---

## Q7. Difference between docker run and docker exec?

docker run creates a new container.

docker exec executes commands inside an existing running container.

---

## Q8. Difference between docker ps and docker ps -a?

docker ps shows running containers.

docker ps -a shows all containers.

---

## Q9. Why did hello-world exit immediately?

Because its purpose was only to print a message and terminate successfully.

---

## Q10. Why is the Nginx container still running?

Because Nginx is a web server that continuously listens for incoming HTTP requests.
